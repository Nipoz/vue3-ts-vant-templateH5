#项目下载  
找到后缀名为.7z的文件，下载后解压，然后用vscode之类的编辑器打开

在终端输入

 ~~
  npm run dev
  
 即可运行项目，axios，跨域代理等都有相关案例，
 
 （
 也可以自行开启后端项目，
 文件夹back就是java文件，新建people数据库（Mysql），导入sql语句后，安装依赖后即可运行，然后配合前端项目提供的接口案例进行对axios的调试
 ）
 
 开箱即用
